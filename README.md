# Dr. Maya Reynolds - Therapy Practice Website

A high-performance, responsive landing page for a licensed clinical psychologist based in Santa Monica, CA. This project was developed as a custom redesign of a therapy practice brand, focusing on high-achieving professionals and creatives.

## ‚ú® Key Features
- **Custom Branding:** Redesigned with a "Healing Earth" color palette (Deep Slate, Soft Sand, and Sage) to evoke grounded resilience.
- **Interactive UI:** Includes a smooth-scroll "Learn More" feature that navigates to detailed therapeutic modalities.
- **Service-Specific Sections:** Dedicated areas for Trauma (EMDR), Anxiety (CBT), and Professional Burnout.
- **Local Asset Integration:** Successfully managed and served local high-fidelity office imagery and professional portraits from the public directory.
- **Fully Responsive:** Optimized for desktop, tablet, and mobile views using Tailwind CSS v4 grid and flexbox systems.
